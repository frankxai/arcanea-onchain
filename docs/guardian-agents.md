# Guardian Agents

> Documentation for ElizaOS-based autonomous Guardian agents.

## Overview

Each of the 10 Arcanea Guardians operates as an autonomous AI agent powered by ElizaOS. These agents have persistent memory, distinct personalities aligned with their Gate, and the ability to perform on-chain actions.

## Agent Capabilities

- **Curate** — Select and highlight exceptional creator works
- **Auction** — Initiate and manage auctions for rare pieces
- **Distribute** — Allocate rewards from Guardian Vaults
- **Evolve** — Update dynamic NFT art based on community interaction
- **Communicate** — Engage with creators and collectors
- **Govern** — Participate in Guardian Council decisions
